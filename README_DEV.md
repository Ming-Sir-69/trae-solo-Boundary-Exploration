# Trae SOLO 能力测试 · 开发文档

## 项目背景

验证 Trae SOLO（AI Agent）在 Code 模式和 MTC（More Than Code）模式下的全栈能力，覆盖前端开发、设备维修、快递分拣、销售记录、仓库管理、运营汇报六大场景。

## 能力维度

| 维度 | 评分 | 说明 |
|------|:---:|------|
| 可视化 | 85 | PPT封面图、雷达图、趋势图、饼图等图表生成 |
| 数据分析 | 90 | Excel数据处理、聚合分析、可视化 |
| 零代码门槛 | 95 | 自然语言驱动，无需手动编码 |
| 多格式处理 | 88 | 22种文件格式预览（docx/xlsx/pdf/csv/md等） |
| 文档生成 | 92 | PPT/XLSX/HTML/Markdown 多格式输出 |

## 6大应用场景

| 编号 | 场景 | 模式 | 模拟数据 |
|:---:|------|------|---------|
| C1 | 前端开发 | Code | 产品JSON |
| C2 | 设备维修 | Code | 设备日志CSV |
| C3 | 快递分拣 | Code | 快递照片10张 |
| M1 | 销售记录 | MTC | 手写笔记7张 |
| M2 | 仓库管理 | MTC | 入库单PDF 3份 |
| M3 | 运营汇报 | MTC | 用户反馈Excel |
